## ğŸš€ Getting Started

### Prerequisites

* [Node.js](https://nodejs.org/) installed on your machine.

### Installation

1. Clone the repository:
```bash
git clone https://github.com/hasan4426/restaurant-reservation-and-table-management-system.git

```


2. Navigate to the project directory:
```bash
cd restaurant-reservation-and-table-management-system

```


3. Install dependencies:
```bash
npm install

```



### Running the Application

1. Start the server:
```bash
npm start

```


*(Or run `node server.js` if nodemon is not used)*.
2. Open your browser and navigate to:
```
http://localhost:3000

```
